VLM 바운딩 박스 · 부피 추정

로컬 Ollama 비전 모델이 각 객체에 대해 extent_xyz(상대 크기)와 2D 투영을 내립니다. 미리보기에는 그 비율로 만든 직육면체를 등각 투영한 와이어프레임을 검출 박스 안에 맞춰 그립니다(실제 3D 재구성은 아님). 부피 지수는 ex·ey·ez이며, 실측(cm) 보정은 표에서 즉시 반영됩니다.

업로드

환경 변수 기본값을 바꾸려면 .env의 OLLAMA_MODEL을 수정하세요.

보정 (선택)

분석 후 재실행 없이 값만 바꿔도 표의 부피(㎤)가 즉시 갱신됩니다. 기준은 선택한 박스의 extent_xyz 한 축과 실측(cm)입니다.

설정 참고

  • 최대 변 리사이즈: {{ max_image_edge }} px
  • Ollama 주소는 서버 환경 변수 OLLAMA_HOST

미리보기

분석 후 extent 비율의 직육면체를 등각 투영한 8점·12선 와이어를 검출 박스 안에 맞춰 겹칩니다(실제 3D 복원은 아님).
